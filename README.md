Design and implement a Customer Relationship Management (CRM) System using Data Structures in C that manages customers, sales quotes, orders, and contracts efficiently.

The system :

Stores and manages customer details using linked lists.

Handles sales quotes and mark them as accepted or pending.

Use a queue structure for managing and processing orders in FIFO order.

Track and “send” contracts based on accepted quotes.

Display analytics including total customers, quotes, orders, and profit estimation
